☁️ Google Drive — Sincronización
Sincroniza tu colección entre dispositivos
Tu colección se guardará en un archivo relojify-sync.json en tu Google Drive. Accesible desde cualquier dispositivo con tu cuenta Google.
1. Ve a console.cloud.google.com
2. Crea proyecto → activa Google Drive API
3. Credenciales → OAuth 2.0 → Aplicación web
4. Orígenes autorizados: añade la URL donde usas esta app (o null si la abres como archivo local)
5. Copia el Client ID (termina en .apps.googleusercontent.com)
